# Entornos del proyecto

## Desarrollo
- Propósito: programación y pruebas locales
- Host: localhost
- Acceso: VS Code + terminal

## QA
- Propósito: validación previa del equipo
- Host: rama dev / despliegue de prueba
- Acceso: GitHub Actions + hosting conectado

## Staging
- Propósito: preproducción
- Host: URL de staging
- Acceso: navegador + panel del proveedor

## Producción
- Propósito: versión estable del sistema
- Host: URL pública final
- Acceso: navegador + panel del proveedor
